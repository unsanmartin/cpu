Propiedad distributiva#

d) División#

\[ (a + b) / c = a / c + b / c\]

Acá también se puede aplicar la propiedad distributiva. La propiedad distributiva no se verifica si el divisor está a a la izquierda.

e) Potenciación#

No puede aplicar la propiedad distributiva en las potencias cuando dentro del parentesis hay una suma o una resta. Mientras que si es aplicable si dentro de los paréntesis tenemos una multiplicación o una división.

\[ (a + b)^c \not = a^c + b^c \]

Mientras que si tenemos una multiplicación:

\[ (a * b)^n = a^n * b^n \]

f) Radicación#

La propiedad distributiva NO no se aplica a la suma y a la resta.

\[ \sqrt[c]{a+b} \not = \sqrt[c]{a} + \sqrt[c]{b} \]

Mientras que SI se aplica a la multiplicación y la división:

\[ \sqrt[c]{a*b} = \sqrt[c]{a} * \sqrt[c]{b} \]

Siempre que la raíz sea un par vamos a tener dos resultados posibles, uno positivo y otro negativo. Si la raíz es impar nos sólo tendremos un número positivo.

Dentro del conjunto de números naturales no se puede sacar la raíz de un número.

III) Números racionales#

Son aquellos que se pueden expresar como una división de dos números enteros. Los números enteros se lo simboliza con Q.

Factor común en adición:

\[ \frac{a}{b} + \frac{c}{d} = \frac{a*d+c*d}{b*d} \]

Multiplicación:

\[ \frac{a}{b} * \frac{c}{d} = \frac{ac}{bd} \]

\[ \big( \frac{a}{b} \big)^2 = \frac{a}{b}*\frac{a}{b} = \frac{a*a}{b*b} = \frac{a^2}{b^2}\]

División:

\[ \frac{a}{b} / \frac{c}{d} \]

FALTA APUNTE ARRIBA

División:

\[ \frac{a}{b} / \frac{c}{d} = \frac{a}{b}*\big(\frac{c}{d}\big)^{\smash{-1}} = \frac{a}{b}*\frac{c}{d}\]
