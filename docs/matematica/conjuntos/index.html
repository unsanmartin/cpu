Conjuntos Numéricos

I. Números naturales#

Los números naturales es el conjunto de los número 1, 2, 3, 4, etc. Se simbolizan con N. También tenemos el conjunto de los números naturales ampliado 0, 1, 2, 3, 4, etc. que se simboliza con N.

\[ N = {1,2,3,4, ...} \]

\[ N\scriptscriptstyle 0 = \normalsize {0, 1, 2, 3, 4, ...}\]

II. Números enteros#

(Fuente: bibliografía de clase)

El conjunto de los número enteros se denota con Z, se define de manera que cumpla:

a) \( N\scriptscriptstyle 0 \) esté incluido en Z \( \enspace N\scriptscriptstyle 0 \enspace \subset \normalsize Z \)

b) que al definir las operaciones en Z se conserven las propiedades fundamentales de \( N\scriptscriptstyle 0 \)

\[ \boxed{Z = Z^- \cup{0}\cup Z^+ } \]

c) que las sustracciones con minuendo menor que el sustraendo siempre tengan solución.

III. Números racionales#

Una razón es un cociente. Es una fracción.

\[\frac 1 3 = 0,3333 \]

IV. Números irracionales#

No se pueden expresar como una fracción. Pi es un número irracional.

\[\frac m n con n \]

Número de Euler, también es un números irracional:

\[ e \]

El número de Euler es la base de los logaritmos.

Número Phi es un número irracional, también se lo conoce como número áureo y está muy presente en el diseño, el arte y la arquitectura:

\[ \phi = 1,47 \]

V. Números reales#

Es el conjunto de todos los números anteriores.

Practicar facciones.

\[\ (m_n) \]

Algunos símbolos matemáticos y su significado:

\(\in \) pertenencia de conjunto: "pertenece a..."

\(\forall \) Cuantificador universal: para todos.

Propiedad distributiva#

c) Multiplicación#

Con respecto a la adición:

\(\forall \) a, b, c \(\in \) \( N\scriptscriptstyle 2 \) se verifica que: (a + b)*c = a * c + b * c (a la derecha)

\(\forall \) a, b, c \(\in \) \( N\scriptscriptstyle 2 \) se verifica que: c * (a + b) = c * a + c * b (a la izquierda)

Con respectoa la sustracción:

\(\forall \) a, b, c \(\in \) \( N\scriptscriptstyle 2 \) se verifica que: (a - b)*c = a * c - b * c (a la derecha)

\(\forall \) a, b, c \(\in \) \( N\scriptscriptstyle 2 \) se verifica que: c * (a - b) = c * a - c * b (a la izquierda)

d) División#

\[ (a + b) / c = a / c + b / c\]

Acá también se puede aplicar la propiedad distributiva. La propiedad distributiva no se verifica si el divisor está a a la izquierda.

e) Potenciación#

No puede aplicar la propiedad distributiva en las potencias cuando dentro del parentesis hay una suma o una resta. Mientras que si es aplicable si dentro de los paréntesis tenemos una multiplicación o una división.

\[ (a + b)^c \not = a^c + b^c \]

Mientras que si tenemos una multiplicación:

\[ (a * b)^n = a^n * b^n \]

f) Radicación#

La propiedad distributiva NO no se aplica a la suma y a la resta.

\[ \sqrt[c]{a+b} \not = \sqrt[c]{a} + \sqrt[c]{b} \]

Mientras que SI se aplica a la multiplicación y la división:

\[ \sqrt[c]{a*b} = \sqrt[c]{a} * \sqrt[c]{b} \]

Siempre que la raíz sea un par vamos a tener dos resultados posibles, uno positivo y otro negativo. Si la raíz es impar nos sólo tendremos un número positivo.

Dentro del conjunto de números naturales no se puede sacar la raíz de un número.

III) Números racionales#

Son aquellos que se pueden expresar como una división de dos números enteros. Los números enteros se lo simboliza con Q.

Factor común en adición:

\[ \frac{a}{b} + \frac{c}{d} = \frac{a*d+c*d}{b*d} \]

Multiplicación:

\[ \frac{a}{b} * \frac{c}{d} = \frac{ac}{bd} \]

\[ \big( \frac{a}{b} \big)^2 = \frac{a}{b}*\frac{a}{b} = \frac{a*a}{b*b} = \frac{a^2}{b^2}\]

División:

\[ \frac{a}{b} / \frac{c}{d} \]

FALTA APUNTE ARRIBA

División:

\[ \frac{a}{b} / \frac{c}{d} = \frac{a}{b}*\big(\frac{c}{d}\big)^{\smash{-1}} = \frac{a}{b}*\frac{c}{d}\]

IV) Números irracionales#

(Tomado de la bibliografía)

Si un número no es decimal exacto ni decimal periódico, no representa a un número racional. Este tipo de números se llaman irracionales, o sea, son aquellos que no pueden expresarse como cociente de dos enteros:

\( \frac{m}{n} \) con n \( \ne \) 0

Al conjunto de los números racionales se lo simboliza con I.

Racionalización de denominadores#

(visto en clase)

La raíz cuadrada en el denominador no puede estar, hay que sacarla con la operación de racionalización de denominadores.

En el ejemplo a) y b) el denominador es un radical único. Ver página 22 de "Elementos de Matemática" de Parrino.

Ejemplos:

a) \( \frac{1}{\sqrt{\smash[b]{2}}} \)

b) \( \frac{1}{\sqrt[3]{16}} \)

c) \( \frac{2}{4-\sqrt[]{5}} \)

d) \( \frac{5}{\sqrt[]{2}+\sqrt[]{7}} \)

Resolución de a):

\[ \frac{1}{\sqrt{\smash[b]{2}}} = \frac{1}{\sqrt[]{2}} * \frac{\sqrt[]{2}}{\sqrt[]{2}} = \frac{\sqrt[]{2}}{\big(\sqrt[]{2}\big)^{\smash{2}}}= \frac{\sqrt[]{2}}{2} \]

Resolución de b):

\[ \frac{1}{\sqrt[3]{16}} = \frac{1}{\sqrt[3]{2^{\smash{4}}}} = \frac{1}{\sqrt[3]{2^{\smash{4}}}}*\frac{\sqrt[3]{2^{\smash{2}}}}{\sqrt[3]{2^{\smash{2}}}} = \frac{\sqrt[3]{2^{\smash{2}}}}{\sqrt[3]{2^{\smash{6}}}} = \frac{\sqrt[3]{2^{\smash{2}}}}{4} \]

En los ejemplos c) y d) el denominador es la suma o la diferencia de dos números.
