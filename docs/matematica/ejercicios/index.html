Ejercicio 6.b.#

Un exponente negativo invierte la fracción sin cambiar el signo de la fracción.

\[ -\big(-\frac{2}{5}\big)^{\smash{-2}} * -\frac{6}{5}\big(-\frac{10}{21}\big) + (-7)^2 : (-7)^3 \]

Resolución del primer término

\[ -\big(-\frac{3}{7})^{\smash{2}} = \frac{9}{49}\]

Segundo término:

\[ -\frac{6}{5} \big(-\frac{10}{21} \big) = \frac{60}{105} = -\frac{4}{7} \]

Tercer término:

\[ \big(-7\big)^{\smash{2}}:\big(-7\big)^{\smash{3}} = \big(-7\big)^{\smash{3}}*\frac{1}{(-7)^{\smash{3}}} = \frac{(-7)^{\smash{2}}}{(-7)^{\smash{3}}} = \frac{1}{(-7)^{\smash{1}}}\]

Sumando todos los términos resueltos:

\[ -\frac{9}{49}-\big(-\frac{4}{7}\big)+\frac{1}{(-7)^{\smash{1}}} = \frac{-9+4*7-7}{49} = \frac{12}{49} \]

\[ \big(-7\big)^{\smash{2}}:\big(-7\big)^{\smash{3}} = \big(-7\big)^{\smash{3}}*\frac{1}{(-7)^{\smash{3}}} = \frac{(-7)^{\smash{2}}}{(-7)^{\smash{3}}} = \frac{1}{(-7)^{\smash{1}}}\]
